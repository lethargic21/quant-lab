# dart-event-study

DART 전자공시에서 이벤트(자사주매입 / 유상증자 / 실적공시)를 감지해 구조화 정보를
추출하고, 이벤트 스터디와 백테스트로 검증한 엔드투엔드 파이프라인.
유니버스는 **2019-01 기준** KOSPI 시총 상위 200 보통주(상폐 종목 포함 — v1.2 survivorship 보정),
기간 2019–2024. **프로젝트 전체 여정 요약은 [notebooks/journey.ipynb](./notebooks/journey.ipynb)**,
결과 상세는 [notebooks/report.ipynb](./notebooks/report.ipynb), 설계 이력은 [PLAN.md](./PLAN.md),
신뢰도 진단·보정 내역은 [RELIABILITY_AUDIT.md](./RELIABILITY_AUDIT.md).

## 결과 요약 (v1.2 — 신뢰도 보정 후)

**1. 공시의 정보력은 실재하고, 보정에도 강건하다** — 시장모형 이벤트 스터디
(표본: 자사주 직접 178 + 신탁 106 / 실적 5,300 / 유증 139 이벤트 → 방향성 시그널 4,650). t는 `naive → 월클러스터`
병기 (실적공시는 어닝시즌에 날짜가 뭉쳐 naive t 과대 — 시그널 4,294건이 523일에 발생, 최대 하루 48건),
✓/✗는 15개 검정 동시 수행을 반영한 **BH-FDR q값(월CL)** 5% 기준:

| 그룹 | [-1,+1] | [0,+5] | [0,+20] |
|---|---|---|---|
| 자사주 직접+ (N=169) | +3.05% (t 8.3→7.5) ✓ | +3.84% (t 8.0→6.4) ✓ | **+4.88% (t 5.8→4.9) ✓** |
| 자사주 신탁+ (N=104) | +2.56% (t 5.2→4.6) ✓ | +2.19% (q=.06) ✗ | **+4.10% (t 3.1→2.7) ✓** |
| 실적+ (N=2,391) | +1.26% (t 11.6→6.9) ✓ | +0.53% (q=.06) ✗ | n.s. ✗ |
| 실적− (N=1,767) | −0.53% (t −4.4→−2.6, q=.03) ✓ | −0.71% (q=.06) ✗ | n.s. ✗ |
| 유증− (N=49) | −1.88% (q=.12) ✗ | −3.93% (t −3.3→−2.9, q=.02) ✓ | −4.98% (q=.12) ✗ |

FDR 보정 후 살아남는 것: **자사주(직접·신탁) 드리프트, 실적± 공시창 [-1,+1], 유증− [0,+5]**.
신탁계약 자사주(v1.2 확장)는 별도 그룹으로 검증 — 직접취득과 같은 드리프트 패턴을
독립적으로 재현하되 강도는 예상대로 약함 (계약≠매수 확약). 경계 유의였던 실적 [0,+5]는
다중검정 보정에서 탈락 — 탈락한 대로 기록한다.
(유증+ N=22는 표본 유보. 실적 QoQ 폴백 68건은 계절성 혼입 우려로 시그널 제외 — 민감도 실질 동일)

**1b. Placebo·비모수 검정이 결론을 뒤집지 않는다 (그리고 한 건은 뒤집는다, 정직하게)** —
naive/클러스터 t 위에 검정 배터리(BMP·Corrado 순위·일반화 부호·CAAR 부트스트랩·placebo
1,000회, [caar_tests.py](./src/dart_event_study/analysis/caar_tests.py))를 얹어 재검증:

| 그룹·윈도우 | 실제 CAAR | placebo 귀무평균 | placebo p (q) | BMP p | Corrado p | 판정 |
|---|---|---|---|---|---|---|
| 자사주 직접 [0,+20] | +4.88% | −0.21% | 0.001 (.002) | <.001 | <.001 | **강건** |
| 자사주 신탁 [0,+20] | +4.10% | −0.28% | 0.001 (.002) | <.001 | .002 | **강건** |
| 실적+ [0,+20] | +0.10% | **+0.11%** | 0.56 (n.s.) | .002 | .97 | **효과 아님** |
| 유증− [0,+5] | −3.93% | −0.02% | 0.007 (.011) | .14 | .002 | **효과 실재** |

- **자사주 드리프트(직접·신탁)는 전 검정 통과.** 랜덤 가짜 이벤트일로는 재현되지 않고
  (placebo 귀무평균 ≈ 0), BMP·Corrado·부호검정이 모두 같은 방향으로 유의. 헤드라인 결과가
  가장 엄격한 sanity check를 넘긴다.
- **실적+ [0,+20]은 placebo가 파라미터 위양성을 걷어낸다.** BMP·부호검정은 유의(p<.01)로
  보이지만 이는 귀무를 CAAR=0으로 놓기 때문 — 대형 성장주는 아무 20일 창이나 조금 오른다.
  placebo 귀무평균이 +0.11%로 실제 +0.10%와 사실상 동일(placebo p=0.56, Corrado p=0.97) →
  **이벤트 효과가 아니라 기저 드리프트.** 기존 "[0,+20] n.s." 결론을 견고하게 재확인.
- **유증−은 비모수·placebo가 저검정력 BMP가 놓친 효과를 잡는다.** N=49·두꺼운 꼬리 탓에
  BMP는 비유의(p=.14)지만 Corrado·부호·부트스트랩·placebo가 모두 유의(placebo q<.05) —
  희석 악재는 실재. 파라미터/비모수 검정을 함께 두는 이유가 여기서 드러난다.
- 요약: **placebo는 자사주를 확증하고, 실적 [0,+20]의 겉보기 유의를 기각하며, 유증−을
  구제한다.** 결론 방향은 유지되고 근거만 두꺼워졌다.

**2. 체결가능한 초과수익은 보정 사다리를 거치며 소멸한다** — 자사주 H=5(유일 후보)의 여정:

| 보정 단계 | net 연수익 | Sharpe |
|---|---|---|
| v1.1: 생존자 스냅샷 유니버스 + 고정 세율 0.20% | +5.1% | 0.21 |
| + [2] 2019년 기준 유니버스·상폐 처리 (승자 편향 제거) | +3.8% | 0.17 |
| + [3] 시행일별 실효 세율(0.30→0.18%) + 수수료 | **+2.4%** | **0.11** |
| KOSPI 벤치마크 | +3.1% | 0.16 |

- **통계적 유의성 [5]**: 블록 부트스트랩 95% CI [−0.57, +0.97] (산술 Sharpe 0.22 기준),
  P(Sharpe≤0)=29.5%. **Deflated Sharpe = 7%** — 시도한 조합 중 우연 기대 최대
  Sharpe에 못 미침. **우연과 구분 불가.**
- **신탁 자사주도 동일 판정**: 신탁 H=20이 net 연 +12.2%, Sharpe 0.46으로 새로운 최선
  조합이지만, 이는 시도 조합이 20개로 늘어난 뒤의 사후 선택 — 부트스트랩 CI
  [−0.30, +1.49], **DSR 16.8%**로 역시 우연과 구분 불가. 좋아 보이는 새 조합이 나올
  때마다 같은 잣대를 들이대는 것이 이 프로젝트의 규율이다.
- **서브기간 [6]**: 자사주 H=5는 2019–21 연 −7.2% (KOSPI +14.3%) vs 2022–24 **+16.3%**
  (KOSPI −7.3%) — 성과가 후반 국면(밸류업 담론기)에 집중. 국면 의존적.
- 실적 시그널: 회전율 43~55×/년, 실효 비용 반영 후 net 연 −20% — 거래 불가 확정.

**결론 (v1.2)**: 공시의 정보력(자사주 드리프트 +4.9%, 공시창 반응)은 유니버스·통계 보정을
전부 통과할 만큼 강건하다. 그러나 v1.1에서 "간신히 생존"으로 보였던 체결가능 초과수익은
**상당 부분 승자 편향(유니버스 look-ahead)과 비용 과소평가의 산물**이었고, 보정 후에는
벤치마크를 밑돌며 우연과 구분되지 않는다. 이 보정 사다리의 정량화가 이 프로젝트의 기여다.

## 방법론

- **이벤트 추출**: OpenDART 구조화 API 우선 (`tsstkAqDecsn`, `piicDecsn`, `stockTotqySttus`),
  잠정실적만 공시 원문(`document.xml`) 표 파싱. 서프라이즈는 공시 안의 전년동기 수치로
  YoY 계산 (외부 컨센서스 불필요 → look-ahead 원천 차단). 전년동기 결측 시 QoQ를 참고
  피처로 계산하되(`surprise_basis` 기록) 시그널에서는 제외 (계절성 혼입 방지).
- **방향/강도 룰** ([config/settings.yaml](./config/settings.yaml)):
  자사주 직접취득 = +1, 강도 = 취득예정주식수/발행주식총수(이벤트일 이전 공개된 정기보고서 기준).
  신탁계약 자사주 = +1, 강도 = 계약금액/(접수 전일종가×발행총수) — 신탁 공시는 금액 기반(실측).
  유증 = 배정방식 기본점수(주주배정·일반공모 −1, 제3자 0) + 자금용도 보정(채무상환 −0.5,
  시설·타법인취득 +0.5), 강도 = 희석률(신주/기존주식수 — 공시 필드만 사용).
- **유니버스 (v1.2)**: 2019-01 프록시 시총(주식수 × 2019-01 수정종가) 상위 200 보통주.
  기간 내 상폐 종목 11종 포함 — 손실형(감사의견) 상폐는 마지막 가격 대비 30% 청산 할인,
  합병·완전자회사화는 무할인 청산. `selection: snapshot_current`로 v1 방식 재현 가능.
- **타이밍 (look-ahead 방지)**: 접수시각이 데이터에 없어(실측) 전 건 **익영업일 종가 체결**의
  보수적 가정. 거래정지일(거래량 0)은 다음 거래 가능일로 밀기 (최대 5거래일, 초과 시 포기).
- **백테스트**: 동시 포지션 균등가중, |Δw| 기반 비용 — 거래세는 시행일별 스케줄
  (2019 0.30/0.25% → 2024 0.18%, 매도측) + 슬리피지 0.10% + 수수료 0.015% (편도),
  수정주가, KOSPI 벤치마크. 엔진은 [shared/backtest](../../shared/quantlab_shared/backtest).
- **통계**: CAR 유의성 = 일/월 클러스터 로버스트 SE(CR1) + BH-FDR q값
  ([analysis/event_study.py](./src/dart_event_study/analysis/event_study.py)).
  추가 검정 배터리 ([analysis/caar_tests.py](./src/dart_event_study/analysis/caar_tests.py)):
  **BMP**(표준화 단면 t — event-induced variance 보정), **Corrado 순위검정**·**일반화
  부호검정**(비모수), **CAAR event-level 부트스트랩 CI**, **placebo/permutation**(랜덤
  가짜 이벤트일 1,000회 → CAAR 귀무분포). 백테스트 Sharpe = 블록 부트스트랩 CI +
  Deflated Sharpe(시도 조합 수 보정).
- 유닛테스트 105개 (타이밍·파서·방향룰·시장모형·클러스터 t·FDR·BMP·Corrado·부호검정·
  placebo·엔진·상폐·정지·비용 — 네트워크 불필요).

## 재현 방법

```bash
# 레포 루트에서
uv sync
cp .env.example .env          # OPENDART_API_KEY 입력 (https://opendart.fss.or.kr)

# 파이프라인 — 단계별 CLI (모든 API 응답은 data/에 캐시, 재실행 시 재크롤링 없음)
cd projects/dart-event-study
alias des="uv run --project . python -m dart_event_study"

des --list                # 실행 가능한 단계 전체 목록
des collect               # OpenDART 공시 수집
des extract-events        # 이벤트 3종 감지·구조화 추출
des build-signals         # 방향/강도 시그널 + 체결시점 매핑
des event-study           # AAR/CAR + 클러스터 t + FDR
des caar-tests            # BMP·Corrado·부호·부트스트랩·placebo
des backtest              # 포트폴리오 백테스트 vs KOSPI
des significance          # Sharpe 부트스트랩 + Deflated Sharpe
des news-collect          # 뉴스 수집 (어텐션 레이어)
des attention             # 어텐션 삼분위 드리프트
des interaction           # 이벤트 × 어텐션 상호작용 회귀

# 결과 노트북
uv run --project projects/dart-event-study jupyter nbconvert --to notebook --execute --inplace \
  projects/dart-event-study/notebooks/report.ipynb
```

full 모드(200종목) 첫 실행은 API ~1.5만 호출 / 30~60분 (OpenDART 일한도 2만).
개발 시 `--mode debug`(5종목)로 전환 — **yaml을 고치지 않고** 메모리에서만 오버라이드된다:

```bash
des event-study --mode debug            # 유니버스 모드 오버라이드
des caar-tests --config-dir path/to/cfg # 설정 디렉터리 통째로 교체
```

기존 모듈 직접 실행(`python -m dart_event_study.analysis.event_study`)도 그대로 동작한다 —
통합 CLI는 그 위에 얹은 추가 계층이다(토스 스케줄러가 모듈 경로를 직접 호출하고 있어 유지).

검증(린트·테스트)은 API 키·네트워크 없이 레포 루트에서 바로 돌아간다 — CI와 동일한 명령:

```bash
uv run --frozen ruff check projects shared
uv run --frozen pytest projects/dart-event-study/tests -q
```

## 한계

- **유니버스 프록시**: 2019-01 시총을 "현재(상폐 시점) 주식수 × 2019-01 수정종가"로 근사 —
  이후 유상증자를 한 종목은 2019 시총이 과대평가됨. 기간 중 신규 상장/급성장 종목은
  미편입(시작 시점 고정) — 보수적 방향. 실제 KOSPI200 구성 이력과는 다름 (KRX API 봇차단).
- **접수시각 미확보**: OpenDART는 접수일자만 제공, DART 뷰어·KIND도 과거 시각 미제공(실측)
  → 익영업일 체결 가정. 당일 종가 체결 기회를 버리는 보수적 편향.
- **실적 이벤트 ~13% 방향 판정 불가** (2019~20 원문 미제공 + 표 형식 변형).
- **상폐 청산 할인 30%는 가정** (파라미터) — 실제 정리매매 손실률은 케이스별로 다름.
- **유상증자**: 방향성 시그널 73건 — 배정방식별 세분 시 표본 얇음. 숏 제약(대차·비용) 미반영.
- **컨센서스 부재**: 서프라이즈는 YoY 대용치 — 애널리스트 기대 대비가 아님.
- **수익률은 배당 제외** 가격수익률 (벤치마크 KOSPI 가격지수와 일관). 발행주식총수는
  현재 API 조회값이라 사후 정정(restatement)이 반영됐을 가능성 — 주식총수 정정은 드묾.
- **시장충격 미모델링**: 슬리피지 0.10% 고정 가정 — 민감도: 편도 0.1/0.3/0.5%에서
  자사주 H=5 net 연 +2.4% / −7.5% / −16.5%. 실효 비용이 조금만 높아도 반전되는 취약한 마진.
- Sharpe 관행 차이: 백테스트 표는 기하 연환산/변동성, significance는 산술평균 기준(학계 관행).

## 감성 레이어 (2단계 — 이벤트 × 어텐션)

같은 공시라도 **뉴스 반응이 뜨거운 종목의 드리프트가 다른가.** 먼저 자사주 삼분위 sort
(사전 등록, 자사주 273건, [0,+20], PLAN §11)로 보고, 이어 유증까지 포함한 단면 상호작용
회귀로 승격했다 (아래):

| [0,+20] CAR | 하위 | 중위 | 상위 | 상−하위 |
|---|---|---|---|---|
| 기사 수(attention) 삼분위 | +3.11% | +4.91% | +5.74% | +2.63%p (p=0.13, **비유의**) |
| 제목 감성(룰) 삼분위 | +3.67% | +5.36% | +4.73% | +1.06%p (p=0.54, 무정보) |

**단면 상호작용 회귀 (Phase A — [analysis/interaction.py](./src/dart_event_study/analysis/interaction.py))**:
삼분위 sort를 정식 회귀로 승격하고 **event_type 축을 3종으로 확장**(자사주 직접=기준 / 신탁 / 유증).
유증(139건, 검색어 "유상증자")을 추가해 **방향이 반대인 이벤트**(희석 악재)까지 포함 —
"호재든 악재든 어텐션이 드리프트를 증폭/소멸시키나"를 본다. `CAR ~ Σevent_type + attn +
Σ(event_type:attn) + log시총 + 추정창변동성`, 월클러스터 SE. 어텐션 = log1p(기사 수) z-score.
**수집창이 [-1,+1]이라 사전 관심이 아니라 사후·동시 반응**이므로, 어텐션 창과 겹치는 `CAR[0,+20]`과
겹치지 않는 `CAR[+2,+20]`을 사전에 둘 다 등록해 전부 리포트한다 (N=404, 통제 포함 357):

주요 스펙 = CAR[0,+20] 통제 O (N=357, R²=0.131):

| 항 | 계수 | p | 의미 |
|---|---|---|---|
| `attn` (기준=자사주 어텐션 기울기) | +0.0197 | .061 | 한계적 |
| **`attn:trust`** (상호작용) | +0.0009 | **.96** | 차이 없음 |
| **`attn:rights`** (상호작용) | −0.0209 | **.22** | 차이 없음 |
| `is_rights` (수준) | −0.0650 | **<.001** | 유증은 −6.5%p (희석 악재) |
| `is_trust` (수준) | +0.0107 | .54 | 신탁 ≈ 자사주 |
| `log시총` | −0.0079 | .13 | — |
| `추정창변동성` | −2.28 | .013 | 변동성↑ = 드리프트↓ |

- **상호작용은 여전히 null — 이제 더 견고.** 두 상호작용 항이 4개 스펙 전부에서 비유의
  (`attn:trust` p≥.70, `attn:rights` p≥.22). 방향이 반대인 유증을 넣어도 "어텐션 효과가
  이벤트 타입마다 다르다"는 **증거 없음**. 자사주-only(N=273)에서의 null이 3타입으로 재확인됐다.
- **유증 더미가 강하게 음(−6.5%p, p<.001)** — 모델이 알려진 희석 악재 드리프트를 정확히 잡는다
  (검증 신호). 단 유증 방향은 혼합(제3자 65 / 악재 50 / 호재 24)인데 평균이 강한 음 → **악재
  배정(주주배정·일반공모)이 유증 표본을 지배**함을 시사.
- **어텐션 주효과는 유증을 넣자 약해졌다** — 자사주-only에서 p=.015였던 것이 p=.061로 한계적이 되고,
  겹치지 않는 [+2,+20]에선 사라진다(p=.28). 사전 판정 규칙대로 **겹칠 때만 유의 = 예측이 아니라 동시성**.
- 규모 교락(자사주-only에선 log시총 p=.002)은 유증 추가 후 약해지고(p=.13), 대신 **추정창 변동성**이
  유의한 통제로 등장(변동성 높은 종목일수록 드리프트가 작음).

**결론**: "드리프트가 어텐션에 따라, 그리고 이벤트 타입에 따라 다르게 증폭/소멸되는가"의 답은
호재(자사주)·악재(유증)를 아우른 표본에서 **null**. 유의하게 만들려 스펙을 바꾸지 않았다.
데이터: 네이버 뉴스 날짜필터 크롤링(±1일 창, 자사주 284 + 유증 139건, 유증 기사 중앙값 11·0건 없음,
캐시). 한계: 쿼리 정밀도, 유증 방향 혼합, **실적 5,300건 미수집**(스로틀상 전량 ~10h·밴 위험으로 제외).
