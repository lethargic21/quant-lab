# dart-event-study

DART 전자공시에서 이벤트(자사주매입 / 유상증자 / 실적공시)를 감지해 구조화 정보를
추출하고, 이벤트 스터디와 백테스트로 검증한 엔드투엔드 파이프라인.
유니버스는 **2019-01 기준** KOSPI 시총 상위 200 보통주(상폐 종목 포함 — v1.2 survivorship 보정),
기간 2019–2024. 설계 이력은 [PLAN.md](./PLAN.md), 신뢰도 진단·보정 내역은
[RELIABILITY_AUDIT.md](./RELIABILITY_AUDIT.md), 결과 상세는
[notebooks/report.ipynb](./notebooks/report.ipynb).

## 결과 요약 (v1.2 — 신뢰도 보정 후)

**1. 공시의 정보력은 실재하고, 보정에도 강건하다** — 시장모형 이벤트 스터디
(표본: 자사주 178 / 실적 5,300 / 유증 139 이벤트 → 방향성 시그널 4,544). t는 `naive → 월클러스터`
병기 (실적공시는 어닝시즌에 날짜가 뭉쳐 naive t 과대 — 시그널 4,294건이 523일에 발생, 최대 하루 48건),
✓/✗는 15개 검정 동시 수행을 반영한 **BH-FDR q값(월CL)** 5% 기준:

| 그룹 | [-1,+1] | [0,+5] | [0,+20] |
|---|---|---|---|
| 자사주+ (N=169) | +3.05% (t 8.3→7.5) ✓ | +3.84% (t 8.0→6.4) ✓ | **+4.88% (t 5.8→4.9) ✓** |
| 실적+ (N=2,391) | +1.26% (t 11.6→6.9) ✓ | +0.53% (t 3.8→2.2, q=.06) ✗ | n.s. ✗ |
| 실적− (N=1,767) | −0.53% (t −4.4→−2.6, q=.03) ✓ | −0.71% (t −4.5→−2.2, q=.06) ✗ | n.s. ✗ |
| 유증− (N=49) | −1.88% (q=.13) ✗ | −3.93% (t −3.3→−2.9, q=.02) ✓ | −4.98% (q=.13) ✗ |

FDR 보정 후 살아남는 것: **자사주 전 윈도우, 실적± 공시창 [-1,+1], 유증− [0,+5]**.
경계 유의였던 실적 [0,+5]는 다중검정 보정에서 탈락 — 탈락한 대로 기록한다.
(유증+ N=22는 표본 유보. 실적 QoQ 폴백 68건은 계절성 혼입 우려로 시그널 제외 — 민감도 실질 동일)

**2. 체결가능한 초과수익은 보정 사다리를 거치며 소멸한다** — 자사주 H=5(유일 후보)의 여정:

| 보정 단계 | net 연수익 | Sharpe |
|---|---|---|
| v1.1: 생존자 스냅샷 유니버스 + 고정 세율 0.20% | +5.1% | 0.21 |
| + [2] 2019년 기준 유니버스·상폐 처리 (승자 편향 제거) | +3.8% | 0.17 |
| + [3] 시행일별 실효 세율(0.30→0.18%) + 수수료 | **+2.4%** | **0.11** |
| KOSPI 벤치마크 | +3.1% | 0.16 |

- **통계적 유의성 [5]**: 블록 부트스트랩 95% CI [−0.57, +0.97] (산술 Sharpe 0.22 기준),
  P(Sharpe≤0)=29.5%. **Deflated Sharpe = 7%** — 시도한 16개 조합 중 우연 기대 최대
  Sharpe(0.82)에 못 미침. **우연과 구분 불가.**
- **서브기간 [6]**: 자사주 H=5는 2019–21 연 −7.2% (KOSPI +14.3%) vs 2022–24 **+16.3%**
  (KOSPI −7.3%) — 성과가 후반 국면(밸류업 담론기)에 집중. 국면 의존적.
- 실적 시그널: 회전율 43~55×/년, 실효 비용 반영 후 net 연 −20% — 거래 불가 확정.

**결론 (v1.2)**: 공시의 정보력(자사주 드리프트 +4.9%, 공시창 반응)은 유니버스·통계 보정을
전부 통과할 만큼 강건하다. 그러나 v1.1에서 "간신히 생존"으로 보였던 체결가능 초과수익은
**상당 부분 승자 편향(유니버스 look-ahead)과 비용 과소평가의 산물**이었고, 보정 후에는
벤치마크를 밑돌며 우연과 구분되지 않는다. 이 보정 사다리의 정량화가 이 프로젝트의 기여다.

## 방법론

- **이벤트 추출**: OpenDART 구조화 API 우선 (`tsstkAqDecsn`, `piicDecsn`, `stockTotqySttus`),
  잠정실적만 공시 원문(`document.xml`) 표 파싱. 서프라이즈는 공시 안의 전년동기 수치로
  YoY 계산 (외부 컨센서스 불필요 → look-ahead 원천 차단). 전년동기 결측 시 QoQ를 참고
  피처로 계산하되(`surprise_basis` 기록) 시그널에서는 제외 (계절성 혼입 방지).
- **방향/강도 룰** ([config/settings.yaml](./config/settings.yaml)):
  자사주 = +1, 강도 = 취득예정주식수/발행주식총수(이벤트일 이전 공개된 정기보고서 기준).
  유증 = 배정방식 기본점수(주주배정·일반공모 −1, 제3자 0) + 자금용도 보정(채무상환 −0.5,
  시설·타법인취득 +0.5), 강도 = 희석률(신주/기존주식수 — 공시 필드만 사용).
- **유니버스 (v1.2)**: 2019-01 프록시 시총(주식수 × 2019-01 수정종가) 상위 200 보통주.
  기간 내 상폐 종목 11종 포함 — 손실형(감사의견) 상폐는 마지막 가격 대비 30% 청산 할인,
  합병·완전자회사화는 무할인 청산. `selection: snapshot_current`로 v1 방식 재현 가능.
- **타이밍 (look-ahead 방지)**: 접수시각이 데이터에 없어(실측) 전 건 **익영업일 종가 체결**의
  보수적 가정. 거래정지일(거래량 0)은 다음 거래 가능일로 밀기 (최대 5거래일, 초과 시 포기).
- **백테스트**: 동시 포지션 균등가중, |Δw| 기반 비용 — 거래세는 시행일별 스케줄
  (2019 0.30/0.25% → 2024 0.18%, 매도측) + 슬리피지 0.10% + 수수료 0.015% (편도),
  수정주가, KOSPI 벤치마크. 엔진은 [shared/backtest](../../shared/quantlab_shared/backtest).
- **통계**: CAR 유의성 = 일/월 클러스터 로버스트 SE(CR1) + BH-FDR q값.
  백테스트 Sharpe = 블록 부트스트랩 CI + Deflated Sharpe(시도 조합 수 보정).
- 유닛테스트 66개 (타이밍·파서·방향룰·시장모형·클러스터 t·FDR·엔진·상폐·정지·비용 — 네트워크 불필요).

## 한계

- **유니버스 프록시**: 2019-01 시총을 "현재(상폐 시점) 주식수 × 2019-01 수정종가"로 근사 —
  이후 유상증자를 한 종목은 2019 시총이 과대평가됨. 기간 중 신규 상장/급성장 종목은
  미편입(시작 시점 고정) — 보수적 방향. 실제 KOSPI200 구성 이력과는 다름 (KRX API 봇차단).
- **접수시각 미확보**: OpenDART는 접수일자만 제공, DART 뷰어·KIND도 과거 시각 미제공(실측)
  → 익영업일 체결 가정. 당일 종가 체결 기회를 버리는 보수적 편향.
- **실적 이벤트 ~13% 방향 판정 불가** (2019~20 원문 미제공 + 표 형식 변형).
- **상폐 청산 할인 30%는 가정** (파라미터) — 실제 정리매매 손실률은 케이스별로 다름.
- **유상증자**: 방향성 시그널 73건 — 배정방식별 세분 시 표본 얇음. 숏 제약(대차·비용) 미반영.
- **컨센서스 부재**: 서프라이즈는 YoY 대용치 — 애널리스트 기대 대비가 아님.
- **수익률은 배당 제외** 가격수익률 (벤치마크 KOSPI 가격지수와 일관). 발행주식총수는
  현재 API 조회값이라 사후 정정(restatement)이 반영됐을 가능성 — 주식총수 정정은 드묾.
- **시장충격 미모델링**: 슬리피지 0.10% 고정 가정 — 소형주 이벤트 직후 실효 비용은 더 클 수 있음.
- Sharpe 관행 차이: 백테스트 표는 기하 연환산/변동성, significance는 산술평균 기준(학계 관행).
