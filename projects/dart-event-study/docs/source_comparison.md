## 후보별 코멘트

### 토스증권 — 유일한 실질 후보이나 핵심 미해결
- **robots**: `*`에 `Allow: /`(`/_ul/`만 금지), AI/Claude 전용 차단 **없음**. Gate 1 통과.
- **웹접근**: 커뮤니티 웹 URL 존재(`/stocks/A005930/community`)하고 브라우저에서 렌더됨. 단
  **완전 클라이언트 렌더 SPA**(정적 HTML은 22.6KB 셸, 종목 무관 동일·게시글 미임베드) →
  프로그래밍 수집은 웹 API 필요. 비크롬 브라우저엔 "지원하지 않는 브라우저" 경고.
- **밀도/활성도**: 커뮤니티 MAU 220만(공개 보도), 렌더된 삼성 커뮤니티에 실사용자 글 활발.
  팍스넷을 정성적으로 압도하나 **일평균 글 수를 정량 집계하지 못함**(SPA 상호작용 제약).
- **댓글**: 글별 좋아요·댓글 수 표시(예: 댓글 5) — 댓글 존재.
- **오염도**: 렌더된 표본은 개미 잡담·한탄 위주로 **리딩방 도배가 거의 없어 보임**(팍스넷보다 깨끗).
  단 소표본 관찰.
- **⚠️ 미해결(연구 성립 관건)**: 커뮤니티 목록이 **인기순 기본이고 게시글 시각을 표시하지 않음**.
  초/분 단위 타임스탬프를 상세페이지나 웹 API에서 얻을 수 있는지 **미확인**. 이게 안 되면
  장후 구간 분리 불가 → 팍스넷과 같은 이유로 탈락. **다음 단계 1순위 확인 항목.**

### 증권플러스 (두나무) — 웹 게시판 부재
- robots: `Allow: /`, `Disallow: /api`. 홈은 SPA 셸(4.3KB). **sitemap 4,565개 URL 전부 종목정보
  (`/m/stocks`)이고 토론/게시판 URL 0개.** 종목토론은 앱 기능이며 웹 미노출. 콘텐츠는 `/api`
  (robots 차단)에서 로드 → robots 준수 시 수집 불가. **탈락.**

### 카카오페이증권 — 앱 전용
- `securities.kakaopay.com` 웹서버 미응답(ConnectionError), 웹 플랫폼 부재 확인. 커뮤니티는
  앱 내부. 앱 API 리버스엔지니어링은 제약상 금지. **탈락.**

### 다음 금융 — robots 모호, 보류
- `finance.daum.net/robots.txt` = **404(부재)**. 그러나 부모 `www.daum.net`은 `*`에 `Disallow: /`
  (홈·ads.txt만 허용) 전면금지. robots는 호스트별이라 기술적으론 서브도메인 무제한이나, 카카오/다음
  포털의 명시적 전면금지 기조와 충돌. 게다가 다음 금융은 과거 종목토론 기능을 축소·정리한 것으로
  알려져 **게시판 존재 자체가 불확실**. **모호하므로 요청을 보내지 않고 보류 — 사용자 판단 필요.**

### 디시인사이드 주식갤러리 / 씽크풀 — Anthropic/Claude 명시 차단
- 두 사이트 robots.txt가 `ClaudeBot`, `anthropic-ai`, `Claude-Web`, `Claude-User`,
  `Claude-SearchBot`를 명시적으로 `Disallow: /`. 이 정찰은 Claude가 수행하므로, 커스텀 UA로
  우회하는 것은 명시 차단 회피에 해당. **원칙(우회 금지·정직)에 따라 접근 불가로 기록, 요청 0건.**
  (`*`에는 Allow인 곳도 있으나, 운영자가 Anthropic 시스템을 명시 배제한 의사를 존중.)

## 종합

### 1) 추천 1순위 — 없음 (2차 정찰 후 갱신)
1차에서 토스를 조건부 1순위로 봤으나, **2차 정찰에서 탈락**(아래 "토스 2차 정찰"): 데이터가
WebSocket으로만 오고 허용된 관찰 범위로는 15:30 경계용 시각을 확인할 수 없으며, 확인하려면
제약상 금지된 WS 가로채기/리버스가 필요. → **접근 가능·타임스탬프 확인된 소스는 팍스넷이 유일**하나
밀도가 치명적으로 낮다(장후 중앙 7글). **현재 조건(합법·robots준수·비RE)에서 팍스넷을 능가하는
소스는 확인되지 않음.**

### 2) 탈락 후보와 사유
- 증권플러스: 웹에 종목토론 미노출 + 콘텐츠 `/api`(robots 차단).
- 카카오페이증권: 웹 플랫폼 부재(앱 전용).
- 디시·씽크풀: Anthropic/Claude 크롤러 명시 차단.
- 네이버 금융: `*` 전면 `Disallow: /`.
- 다음 금융: robots 모호 + 게시판 존재 불확실(보류).

### 3) 팍스넷 대비 개선폭
- 토스가 타임스탬프 관문을 통과한다면, MAU·활성도로 볼 때 밀도는 **정성적으로 1~2 자릿수 배**
  개선 가능(정량 미확보 — 과장 방지 위해 배수 단정 안 함). 오염도도 팍스넷 33%보다 낮아 보임.
- 통과 못 하면 **현재까지 모든 접근가능 소스 중 타임스탬프+정적수집이 확인된 곳은 팍스넷뿐**이며,
  팍스넷의 밀도 한계(장후 중앙 7글)가 그대로 남는다.

### 4) 남은 불확실성
- **[최우선] 토스 게시글 타임스탬프 해상도/노출 위치** — 상세페이지·API 미확인.
- 토스 데이터 공급 API의 서브도메인·robots 정책 미확인.
- 토스 일평균 글 수 정량치 미확보(SPA 상호작용 제약으로 이번 정찰서 집계 불가).
- 다음 금융 robots 모호성 — 진행 여부는 사용자 결정 사안.
